# Help a User Set Up a Custom Domain

## Wording rules
- Use Name / Host for the left-hand field.
- Use Content / Value for TXT values.
- Use Target for CNAME records.
- Mention some providers use @ for the root domain.

## Flow
1. Add the domain in PlugLayer.
2. Show the exact TXT and route records.
3. Explain where each value goes in the DNS UI.
4. Tell the user: after you add the records, tell me you've added them and I'll verify and continue.
